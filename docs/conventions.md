# Code Conventions

## Custom CSS Classes (from custom.css)

- .section-title
- .section-body
- .experience-item
- .item-title / .item-subtitle
- .skill-tag
- .testimonial-item
- .contact-item
- .contact-icon

## Naming Pattern

- Use kebab-case for classes
- Prefix section-specific classes with section name (e.g., `.experience-item`)
- Use semantic HTML5 tags where possible

## Spacing Consistency

- Sections: `w3-padding-24` or `w3-padding-32`
- Items within sections: `w3-margin-top` (standard)
